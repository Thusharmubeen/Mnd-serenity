Two distinct algorithms make up our method's customization component. Specific real-time environment adjustments are provided by the second, while the first selects the specific environment that will be displayed. Both algorithms use dynamic anxiety data and static personality data as inputs when they make decisions. These customization algorithms are intended to be implemented using two distinct methods..Because human preference does not change significantly over time, the environment decision is made via semantic reasoning. In real time adaptations, or the second approach, are carried out by contextual multi-armed bandit learning ,a reinforcement learning subgroup.

Because the algorithm for choosing a relaxing setting is based on the manual construction of rules that define human behaviour its creation necessitates a high level of subject knowledge. It expresses, in particular, how a user's preferences for a certain environment are influenced by specific answers. Applying a semantic reasoner to the model's data and the manually defined rule comprises the algorithm.

A self-learning approach is selected for the real-time adaption algorithms. More specifically, the ideal choice is contextual multi-armed bandits. These learning algorithms attempt to select an action from a variety of options after receiving some context, in this case personality and anxiety data. In this example, the action is changing a parameter's value in the virtual reality environment. Contextual bandit agents gain experience during time by being rewarded for their actions. More specifically, in this instance, the physiological reaction can be used to determine the reward. Making wise decisions will lead to less stress and more comfort, whereas making poor decisions has the opposite impact.
